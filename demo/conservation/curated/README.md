## What's tailored per variant

- **Headline title** — matches the posting
- **Summary** — leads with the experience most relevant to the posting (forestry vs. range vs. wildlife vs. crew leadership)
- **Skills** — same skill set, reordered so role-relevant items come first
- **Certifications** — ordered by relevance (e.g. S-212 leads for chainsaw-heavy roles; pesticide cert leads for invasives-heavy roles)
- **Bullets** — same underlying achievements, lightly reordered/rephrased to surface the relevant angle (e.g. "vegetation surveys across pinyon-juniper" vs. "chainsaw work in pinyon-juniper" — both true, different emphasis)

The base facts (jobs, dates, employers, education) are constant across variants.
